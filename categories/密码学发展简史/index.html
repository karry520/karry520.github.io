密码学发展简史

2020-04-04 密码学 Comments

古典密码

古代密码

从由人类以来到1800年，密码设计与分析被当作一门艺术。这一时期的密码学专家常常是凭直觉和信念来进行密码设计和分析，而不是靠推理证明。数据的保密基于加密算法的保密。著名密码算法：

  1. 500B.C.， 古斯巴达“天书”密码
  2. 205-123 B.C.，古希腊人棋盘密码
  3. 50 B.C.， 古罗马恺撒密码
  4. 16世纪，维吉尼亚的密码

近代密码

从1800到1949年，密码机的迅速发展，越来越多的数学家加入密码队伍。

  1. 1795年，杰弗逊圆盘
  2. 1914年，美陆军和海军的M-128-T4
  3. 1928年，德国的Enigma密码机
  4. 1926年，Kryha密码机
  5. 1936年，瑞典的哈格林发明的Haglin密码机

1949年之前，古典密码特点是，密码学还不是科学，而是艺术；出现一些密码算法和加密设备；出现密码算法设计和基本手段，数据的保密基于加密算法的保密。1883年Kerckhoffs第一次明确提出了密码编码的原则

加密算法就建立在算法的公开不影响明文和密钥的安全，即密码算法的安全性仅依赖于对密钥的保密。

这一原则成为判定密码强度的衡量标准，也成为古典密码和现代密码的分界线之一。

现代密码

现代密码I（1949-1976）

  1. 1949年，Shannon发表“The Communication Theory of Secret Systems”

    • 定义理论安全性，提出扩散和混淆原则
    • 奠定了密码学的理论基础
    • 艺术$\to$科学

  2. 1976年的Kahn的“The Codebreakers”

  3. 1971-73年IBM的Feistel等的几篇技术报告

  4. Lucifer$\to$DES

数据的安全基于密钥而不是算法的保密

现代密码II（1976-1994）

  1. 1976年Diffie&Hellman的“New Directions in Cryptography”提出了公钥密码的概念
  2. 1977年Rivest, Shamir&Adleman提出了RSA公钥算法
  3. 1977年，DES成为了第一代公开的、完全说明细节的商业级密码标准
  4. 90年代逐步出现椭圆曲线等其他公钥算法

公钥密码部分解决了对称密钥密码算法密钥共享和密钥管理困难的问题！

现代密码III（1944-）

  1. 1994年，Shor提出量子计算机模型下分解大整数和求解离散对数的多项式时间算法
  2. 2000年，AES正式取代DES成为了新的加密标准
  3. 2006年，第一届后量子密码学国际研讨会召开
  4. 2017年，NIST开始征集后量子密码标准
